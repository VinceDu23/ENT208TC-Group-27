YOLOv8 姿态分析

基于 COCO 17 关键点人体模型，实时计算髋、膝、肩、肘、踝、躯干 6 大关节角度。640×480 输入下达到 30fps 实时推理。

UltralyticsCPU/GPUMJPEG Stream
📐

杠铃轨迹追踪

每 reps 实时绘制杠铃运动路径，与理想轨迹可视化对比。自动计算偏移量（cm），深蹲前移、硬拉远离身体等错误即时检测。

Canvas 2DRep Detection
❤️

BLE 心率 + 加速度计

Web Bluetooth API 直连 Nano 33 BLE / ESP32 等设备，实时读取心率（HR）和 IMU 加速度计数据。5 区心率可视化 + 卡路里估算 + 恢复状态监测。

Web BluetoothGATTIMU
⚡

传感器融合引擎

自动融合 YOLO 关节点 + BLE 加速度计 / 速度数据，弥补单一传感器盲区。当传感器无法连接时无缝降级到仿真模式持续运行。

Data FusionFallback
📊

数据可视化分析

Chart.js 驱动的姿态评分趋势、训练量统计、动作对比雷达图 + 弱点分析建议。训练记录历史支持按动作 / 时间维度过滤。

Chart.js 4.4TrendCompare
🌐

WebSocket 实时通信

FastAPI 后端通过 WebSocket 双向低延迟推送姿态数据流。前端自动重连机制，5s 连接超时 + 5 次指数退避重试，MJPEG 视频流直连 <img> 标签。

工作流程

4 步开启智能训练

1

启动后端服务

终端运行 python pose_server.py，加载 YOLOv8 模型并打开摄像头，WebSocket 服务就绪。

→
2

打开网页应用

浏览器访问 localhost，登录后进入仪表盘。支持 Chrome / Edge，中英双语界面。

→
3

连接设备

一键连接 YOLO 姿态服务 + BLE 心率带 / 加速度计。系统自动融合多源数据，也可在模拟模式下离线演示。

→
4

开始训练

选择动作（深蹲/卧推/硬拉/肩推），点击开始。实时查看关节点角度、杠铃轨迹、心率区间，姿势异常即时报警。
